# develop

Engineering skills for software development workflows.

## Install & Uninstall

```bash
# Install
/plugin install develop@newkayak12-claude-skills

# Uninstall
/plugin uninstall develop@newkayak12-claude-skills
```

## Skills

| Skill | Description |
|-------|-------------|
| `architecture-designer` | Design system architecture, write ADRs, evaluate trade-offs |
| `chaos-engineer` | Design chaos experiments and failure injection frameworks |
| `cli-developer` | Build CLI tools with argument parsing and shell completions |
| `code-documenter` | Generate docstrings, OpenAPI specs, JSDoc, and doc sites |
| `database-optimizer` | Optimize queries and improve PostgreSQL/MySQL performance |
| `frontend-developer` | React, Next.js, Tailwind CSS, TypeScript UI implementation |
| `kotlin-specialist` | Kotlin coroutines, Flow, KMP, Compose, Ktor |
| `mcp-builder` | Build MCP servers in Python or TypeScript |
| `microservices-architect` | Design distributed systems, decompose monoliths |
| `spring-boot-engineer` | Spring Boot 3.x, Security 6, JPA, WebFlux |
| `sql-pro` | Complex queries, window functions, indexing, query plans |
| `sre-engineer` | SLOs, error budgets, incident response, capacity planning |
| `test-driven-development` | TDD workflow before writing implementation code |
| `test-master` | Unit, integration, E2E, performance, and security tests |
