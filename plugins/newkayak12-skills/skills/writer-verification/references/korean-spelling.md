# Korean Spelling Quick Reference

Common Korean spelling mistakes. Load this when Pass 1 flags a potential Korean spelling issue and you need to verify the correct form.

| Wrong | Correct | Note |
|-------|---------|------|
| 됬다 | 됐다 | vowel contraction |
| 어떻해 | 어떡해 | short for "어떻게 해" |
| 왠만하면 | 웬만하면 | unrelated to "왜" |
| ~로써 (role) | ~로서 | 로서=role, 로써=means |
| 있음에도 불구하고 | 있음에도 | redundant |
| ~에 있어서 | ~에서 / ~에 | translated phrasing |
| ~되어진다 | ~된다 | double passive |
| 안되다 | 안 되다 | spacing: 안 + 되다 |
| 몇일 | 며칠 | no "몇일" form exists |
| 설레임 | 설렘 | 설레다 → 설렘 |
| 희안하다 | 희한하다 | |
| 오랫만에 | 오랜만에 | |
| 구렛나루 | 구레나룻 | |
| 낳다/낫다/낮다 | context-dependent | 낳다=give birth, 낫다=better/recover, 낮다=low |
| 맞추다/맞히다 | context-dependent | 맞추다=fit/match, 맞히다=guess correctly |
| 할께요 | 할게요 | ㄹ게요 not ㄹ께요 |
| 않돼 / 않되 | 안 돼 | 안+되다, not 않다 |
| ~ㄹ지도 몰라 | ~ㄹ지도 몰라 | correct, but "~ㄹ지 몰라" is also fine |
| 바램 | 바람 | 바라다 → 바람 (not 바래다) |
| 뵈요 | 봬요 | 뵈어요 → 봬요 (contraction) |
| 되요 | 돼요 | 되어요 → 돼요 |
| 에요/이에요 | context-dependent | consonant ending: 이에요, vowel ending: 예요/에요 |
| 웬지 | 왠지 | 왠지=for some reason (왜+인지) |
| 금새 | 금세 | 금세=soon (금시에) |
| 어의없다 | 어이없다 | 어이=기가 막힘, not 어의 |
