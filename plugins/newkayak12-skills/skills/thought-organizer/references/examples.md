# Additional Examples

## Example 2 — Stream of Consciousness to Core Claim

**Input:**
> "기후변화 글 써야 하는데 뭘 말하고 싶은지 모르겠어. 개인이 뭘 해도 기업이 다 망치는 것 같고, 근데 개인 행동도 중요하긴 하고, 근데 무력감이 드는 게 문제인 것 같고."

**Output:**
```
핵심 주장: 기후 행동의 무력감은 구조적 문제를 개인에게 전가하는 서사에서 비롯된다.

근거:
  - 탄소발자국 개념 자체가 석유 기업의 PR 전략에서 출발
  - 개인 행동의 실질 효과는 구조 변화보다 수십 배 작음

반론:
  - 개인 행동이 사회 규범과 정치적 신호를 만든다

해소 방향:
  - 무력감 → 집단 행동과 정책 참여로 전환하는 프레임 제안

다음 질문: 독자가 글을 읽고 나서 무엇을 하길 원하는가?
```

## Example 3 — Concept Linking (Zettelkasten)

**Input:**
> "자유의지, 결정론, 책임, 이게 다 연결된 것 같은데 어떻게 정리하지?"

**Output:**
```
[001] 결정론: 모든 사건은 선행 원인의 필연적 결과다.
[002] 자유의지 (강한 의미): 원인 사슬 바깥에서 행위를 시작하는 능력 → [001]과 충돌
[003] 양립가능론: 자유의지는 외부 강제 없이 자신의 욕구에 따라 행동하는 것 → [001]과 양립 가능
[004] 도덕적 책임: 행위자가 다르게 행동할 수 있었을 때 성립 → [002]에 의존하면 [001]과 충돌, [003]에 의존하면 유지 가능
[?]   열린 질문: [003]은 진짜 자유의지인가, 아니면 자유의지라는 단어의 재정의인가?
```
