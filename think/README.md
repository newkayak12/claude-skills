# think

Thinking and ideation skills for better decisions and clearer ideas.

## Install & Uninstall

```bash
# Install
/plugin install think@newkayak12-claude-skills

# Uninstall
/plugin uninstall think@newkayak12-claude-skills
```

## Skills

| Skill | Description |
|-------|-------------|
| `brainstorming` | Explore intent and requirements before implementation |
| `decision-maker` | Structured comparison for technology choices and trade-offs |
| `devils-advocate` | Challenge assumptions and stress-test decisions |
| `problem-reframer` | Reframe problems to find better questions, not just more solutions |
| `retrospective` | Structured reflection using KPT, 5 Whys, or timeline formats |
| `thought-organizer` | Organize scattered thoughts into structured outlines |
| `microinteractions` | Design triggers, rules, feedback, and loops for polished product moments (Saffer) |
| `negotiation` | Tactical empathy, calibrated questions, and the Ackerman method (Voss) |
| `first-principles` | Decompose assumptions to irreducible truths and reconstruct from scratch (Aristotle + Musk/Munger) |
