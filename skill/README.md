# skill

Skills for creating, improving, and validating Claude Code skills.

## Install & Uninstall

```bash
# Install
/plugin install skill@newkayak12-claude-skills

# Uninstall
/plugin uninstall skill@newkayak12-claude-skills
```

## Skills

| Skill | Description |
|-------|-------------|
| `skill-creator` | Create new skills, run evals, benchmark performance |
| `skill-validator` | Validate a skill across five quality dimensions |
| `skill-trigger-validator` | Audit and improve skill trigger coverage for natural language and Korean input |
