# self

Self-reflection and personal growth skills for deeper self-understanding.

## Install & Uninstall

```bash
# Install
/plugin install self@newkayak12-claude-skills

# Uninstall
/plugin uninstall self@newkayak12-claude-skills
```

## Skills

| Skill | Description |
|-------|-------------|
| `attachment-style-mirror` | Explore attachment patterns in relationships |
| `ego-state-identifier` | Identify ego states (Parent/Adult/Child) in interactions |
| `examined-life` | Structured Socratic self-examination of values and choices |
| `fear-inventory` | Surface and categorize fears blocking action or growth |
| `flow-antigoal` | Identify anti-goals and flow states to clarify direction |
| `identity-explorer` | Explore core identity, roles, and self-concepts |
| `motivation-explorer` | Uncover intrinsic motivations and energy sources |
| `shadow-persona` | Explore shadow aspects and disowned parts of personality |
| `strength-growth-mapper` | Map strengths and growth edges for development |
| `values-explorer` | Clarify personal values and their hierarchy |
