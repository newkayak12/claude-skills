# agents

Agent orchestration skills for multi-agent workflows.

## Install & Uninstall

```bash
# Install
/plugin install agents@newkayak12-claude-skills

# Uninstall
/plugin uninstall agents@newkayak12-claude-skills
```

## Skills

| Skill | Description |
|-------|-------------|
| `dispatching-parallel-agents` | Dispatch 2+ independent tasks to parallel agents |
| `subagent-driven-development` | Execute implementation plans using subagents with review checkpoints |
