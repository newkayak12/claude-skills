# newkayak12-portfolio

Portfolio and career skills for job search and interview preparation.

## Install & Uninstall

```bash
# Install
claude plugin install newkayak12/claude-skills#newkayak12-portfolio

# Uninstall
claude plugin uninstall newkayak12-portfolio
```

## Skills

| Skill | Description |
|-------|-------------|
| `portfolio-company` | Research a target company before applying or interviewing |
| `portfolio-feedback` | Get structured feedback on your portfolio |
| `portfolio-interview` | Prepare interview answers from your portfolio content |
| `portfolio-jd` | Analyze a job description and align your portfolio to it |
| `portfolio-pattern` | Extract recurring patterns and strengths from your portfolio |
| `portfolio-rewrite` | Rewrite portfolio content for clarity and impact |
