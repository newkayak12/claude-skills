# newkayak12-planning

Planning skills for structured implementation and roadmap execution.

## Install & Uninstall

```bash
# Install
claude plugin install newkayak12/claude-skills#newkayak12-planning

# Uninstall
claude plugin uninstall newkayak12-planning
```

## Skills

| Skill | Description |
|-------|-------------|
| `executing-plans` | Execute a written implementation plan with review checkpoints |
| `roadmap-planning` | Plan a strategic roadmap across prioritization, epics, and sequencing |
