# Leadership Skills

Skills for engineering managers and tech leads: running effective 1-on-1s, leveling engineers, and building high-performing teams.

## Skills

- **1-on-1-manager** — Manager's guide to running high-impact 1-on-1 meetings: agenda design, coaching conversations, growth dialogue
- **1-on-1-ic** — IC's guide to getting value from 1-on-1s: preparation, driving career conversations, upward feedback
- **leveling-manager** — Manager's guide to leveling engineers: rubric design, promotion case writing, calibration conversations
- **leveling-ic** — IC's guide to leveling up: self-assessment, evidence collection, gap analysis, promotion readiness
