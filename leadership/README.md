# leadership

Engineering leadership skills for managers and individual contributors.

## Install & Uninstall

```bash
# Install
/plugin install leadership@newkayak12-claude-skills

# Uninstall
/plugin uninstall leadership@newkayak12-claude-skills
```

## Skills

| Skill | Description |
|-------|-------------|
| `1-on-1-manager` | Run high-impact 1-on-1s: employee-led agendas, coaching questions, growth conversations, feedback |
| `1-on-1-ic` | Get more from your 1-on-1s: prepare agendas, drive career conversations, give upward feedback |
| `leveling-manager` | Design rubrics, write promotion cases, and run calibration conversations as a manager |
| `leveling-ic` | Understand your level gaps, collect evidence, and prepare for promotion conversations as an IC |
